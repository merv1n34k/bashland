# welcome

this is your home directory. you can do anything here — break it, fill it,
delete it. when you refresh the page, you get a fresh one.

## getting started

    ls              list files
    cat README.md   read this file again
    cd lessons      enter the lessons directory
    pwd             show where you are

## what you can use

bash, git, vim, nano, tmux, curl, wget, jq, tree, less, man, htop.

network access is read-only over https — `git clone` and `curl` work, but
nothing can connect back to you.

## limits

each session: 256 MB ram, half a cpu, 64 processes, 64 MB disk in $HOME.
go past any of these and your shell will be killed. that's normal.
