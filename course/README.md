# Welcome

You will learn how to navigate and control the system, how to track
files and changes, and try yourself with reporting a bioinformatics
analysis.


## Tasks

1. Set up a workspace and a git repository.
2. Fetch the Synechocystis sp. PCC 6803 protein FASTA from NCBI
   (assembly GCF_000009725.1).
3. Count proteins, find specific families (psbA, ribosomal), extract
   sequence IDs.
4. Write `report.md` with your findings.
5. Commit your work cleanly, with the raw FASTA gitignored.
